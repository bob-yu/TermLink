TermLink Toolbar Icon Options

Option A - Professional Monoline

All icons use the same dark blue-gray line style. Quiet, durable, and desktop-tool oriented.

Option B - Brand Blue Monoline

Same shapes as Option A, but unified around the app blue. Stronger brand consistency.

Option C - Monoline With Functional Accents

Mostly professional gray, with color only where it helps: connect, disconnect, and serial access.

Option D - Hardware Plate

More hardware-inspired. Small white plates keep icons distinct without the heavy colored-tile look.

Alternative Core Actions - No Plug

Different Add / Connect / Disconnect concepts without plug shapes. These are better for a tool that handles local serial, remote serial, SSH, and Telnet sessions.
